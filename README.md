# Bnuditor

A web app that will soon allow creating, editing and saving levels for Everybody Edits Offline and community mods.

## Tools used

- [Vue](https://vuejs.org/) + [TS](https://www.typescriptlang.org/)
    - A guide for working in projects like these is available [here](https://vuejs.org/guide/typescript/overview).

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
